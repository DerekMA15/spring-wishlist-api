# Wishlist API 🚀

API REST para gerenciamento de lista de desejos, desenvolvida com **Spring Boot 3**, **Java 21** e **PostgreSQL**.

## 🛠 Tecnologias e Conceitos Aplicados
* **Docker & Docker Compose**: Orquestração de containers.
* **JPA Auditing**: Registro automático de data de criação.
* **DTO Pattern**: Isolamento da camada de persistência.
* **Global Exception Handling**: Tratamento padronizado de erros.
* **Swagger (OpenAPI)**: Documentação interativa.
* **Pagination**: Listagem otimizada de recursos.

## 🚀 Como Rodar
1. Certifique-se de ter o Docker instalado.
2. Clone o repositório.
3. No terminal, execute: `docker-compose up --build`
4. Acesse: `http://localhost:8080/swagger-ui/index.html`
